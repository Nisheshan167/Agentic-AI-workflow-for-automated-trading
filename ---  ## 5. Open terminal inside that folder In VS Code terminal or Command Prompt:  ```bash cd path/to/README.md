# Agentic AI Trading Dashboard

This is an interactive Streamlit application for comparing:
- Buy-and-Hold
- Baseline Technical Strategy
- Agentic AI Strategy

## Features
- User-selected stock ticker
- Technical indicators: SMA, RSI, MACD
- Agentic AI workflow with:
  - Market Analysis Agent
  - News Retrieval / Sentiment Agent
  - Decision Agent
  - Risk Agent
- Performance metrics
- Cumulative return comparison
- Optional ChatGPT-based results interpretation

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
